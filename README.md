# C# Fundamentals

A beginner-friendly collection of C# examples demonstrating the core concepts of the language. This repository serves as a quick reference and learning guide for anyone starting with C# and .NET.

## 📚 Concepts Covered

The examples in this repository are divided into small, easy-to-understand files that cover the following topics:

### 1. Variables and Data Types

- **`typesOfData.cs`**: Exploring basic primitives (`int`, `double`, `char`, `bool`, `string`).

### 2. Operators

- **`comparisonOperator.cs`**: Working with `==`, `!=`, `<`, `>`, etc.
- **`logicalOperator.cs`**: Utilizing `&&` (AND), `||` (OR), and `!` (NOT).
- **`ternaryOperator.cs`**: Using the shorthand `condition ? true : false` operator.

### 3. Control Flow

- **`conditions.cs`**: Standard `if`, `else if`, and `else` statements.
- **`switch.cs`**: Handling multiple conditions cleanly with `switch` statements.
- **`loop.cs`**: Iterating with `for`, `while`, and `do-while` loops.

### 4. Arrays

- **`array.cs`**: Declaring and initializing standard C# arrays.
- **`arrayMethods.cs`**: Using built-in methods on arrays like `Array.Sort` and `Array.Reverse`.
- **`stringArray.cs`**: Storing strings inside arrays.
- **`multiDimention.cs` & `multiDimention2.cs`**: Working with 2D and 3D arrays seamlessly.

### 5. Collections

- **`list.cs`**: Dynamic arrays using `List<T>`.
- **`listMethods.cs`**: Manipulating lists with `.Add()`, `.Remove()`, and other built-in list features.
- **`dictionary.cs`**: Key-value pair storage utilizing `Dictionary<TKey, TValue>`.

### 6. Functions (Methods)

- **`functionWithoutReturnValue.cs`**: Void methods that perform actions without returning data.
- **`functionWithReturnValue.cs`**: Methods that calculate and return values.

### 7. Strings

- **`stringMethod.cs`**: Useful string manipulation functions.
- **`stringGrid.cs`**: Formatting operations when working with strings.
